# Intro 

Pour mon journal d'apprentissage, j'ai décidé d'apprendre le React avec pour objectif de m’entraîner à faire un site web de bout en bout. 


Je souhaite apprendre le React car, ayant déjà des bases en HTML, CSS et Javascript, je souhaite approfondir en apprenant une librairie JS. J'ai choisi cette librairie car elle est largement utilisée dans le secteur et aussi au sein de mon entreprise. De plus, son fonctionnement par composants imbriqué me semble particulièrement bien orienté pour faire de l'UI et mettre en place un design système, ce qui est la partie qui m'intéresse.

Je voudrais aussi travailler la programmation dans son ensemble afin d’assimiler les compétences nécessaire pour faire un site vitrine entièrement de mes mains, avec pour objectif d'être en capacité de coder mon portfolio moi même.
